# 🏝️ 코지 아일랜드 — 개발 플랜 / 진행 기록

이 문서는 지금까지의 작업(완료)과 앞으로 할 수 있는 작업(예정)을 정리한다.

## 완료한 작업

### v1 — MVP 코어
- 폴더 스캐폴드, `types.ts`, 전체 `data/*.json`(밸런스/콘텐츠 + `strings.ko`), `registry.ts` 등록.
- 엔진 코어: `Game`(rAF 루프/상태/렌더), `GameState`, `EventBus`, `Camera`, `rng`.
- 탭 이동 + 자동 상호작용 파이프라인, 스태미나, 텐트 취침(0일 때만 + 최대치 +1).
- 채집(나무/바위/덤불/광석), 인벤토리/HUD, 상점(판매·밭 구매).
- 농사(자동 성장·수확·재파종), 동물(닭), 요리, 건물(건설·업그레이드), 퀘스트(15+), 도감.
- 저장(버전 관리·자동저장·오프라인 타이머·리셋), 절차적 오디오 + 설정, 광산 스텁.

### Update 3 — 카메라/지터/자연 지형
- 한 지역이 화면에 꽉 차도록 줌 + 지역 스냅 카메라.
- 단일 월드 트랜스폼으로 오브젝트 지터 제거.
- 체크판 바닥 → 자연 잔디 텍스처, 정적 바다 → 흐르는 물 연출.

### Update 4 — 캐릭터
- 더 아기자기하고 디테일한 플레이어 스프라이트.
- idle(숨쉬기·눈깜빡임) / walk(4프레임) / action(도구 휘두름) / tired(0일 때) 애니메이션.

### Update 5 — main 워크플로
- `claude/new-project-t8sbf5`에서 개발 → 작업마다 `--no-ff`로 main 머지 + 푸시 (이번 세션 한정 규칙).

### Update 6 — 단일 대륙 + 울타리 우리 (최신)
- 다중 섬 + 바다 다리 구조 폐기 → **하나의 땅**(바깥만 바다).
- 내부를 **동일 크기 직사각 우리**로 분할, 사이에 **돌길**, **게이트**로만 진입(울타리는 이동 차단).
- 작물별 밭 분리(밀/당근/딸기/토마토/감자 각각 전용 우리).
- 모든 월드 렌더를 **정사각 픽셀 도트**로 교체(곡선/그라데이션 제거).
- 격자 A\* 길찾기(`utils/pathfind.ts`)로 돌길·게이트 자동 경로.
- `data/worldLayout.json` 재작성 + `game/layout.ts` 신설(우리/게이트/건물위치/앵커 계산).
- 저장 `SAVE_VERSION = 4`로 상향(구조 변경 → 구 세이브 초기화).

### Update 7 — 광산 본구현
- `data/mineLevels.json` 스키마 보강: 층별 `nodeHp`, `staminaCost`, `descendRequirement`, 드롭 가중치.
- `systems/MineSystem.ts` 신설: 층별 노드 절차 배치, 채광 드롭, 이미 캔 노드 기록, 하강 조건 처리.
- `GameState`에 `mineCurrentFloor`, `mineMinedNodes` 추가, 저장 `SAVE_VERSION = 7`로 상향.
- 하단 메뉴 "광산" 진입 버튼 추가, 광산 HUD에서 층 표시/하강/퇴장 가능.
- 광산 루프: 입장 → 채광(스태미나 소모) → 하강 조건 충족 → 더 깊은 층 → 귀환 → 광석 판매 가능.

## 앞으로 할 수 있는 작업 (예정 후보)

> 우선순위는 사용자 요청에 따름. 콘텐츠 추가는 "우리 한 칸 추가" 규칙을 따른다.

- **콘텐츠 확장**: 작물/동물/레시피/건물 우리 추가, 퀘스트 라인 연장.
- **던전**: 데이터/입구만 존재 → 실제 플레이 루프.
- **밸런스 패스**: 첫 5분 루프 체감, 가격/성장시간/스태미나 비용 조정.
- **건물 업그레이드 시각/효과** 다양화, 패시브(`passives.json`) UI 노출.
- **사운드 다양화**: 상황별 BGM 추가 레이어, SFX 보강.
- **모바일 UX 폴리시**: 패널 레이아웃, 안내 화살표, 접근성.
