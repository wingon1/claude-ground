# 🏝️ 코지 아일랜드 — 게임 기획서

> 모바일 세로화면용 2D 픽셀아트 코지 채집·농장 타이쿤 RPG.
> 순수 Canvas 2D 렌더링 + Web Audio 절차적 사운드. 추가 의존성·외부 에셋 없음.
> UI 언어: **한국어**. 데이터(밸런스/콘텐츠)는 전부 `data/*.json`으로 분리.

---

## 1. 핵심 루프

```
채집/수확/채광(스태미나 −1) → 스태미나 0 → 텐트에서 취침
   → 스태미나 전량 회복 + 최대치 +1 → 다시 채집…
```

판매로 골드를 모아 밭·동물·건물·요리를 늘리고, 퀘스트로 초반 동선을 안내한다.
**왕국 레벨 같은 추상 레벨 없음.** 모든 해금은 실제 진행 데이터(골드/누적 횟수/건물/최대 스태미나/퀘스트)로만 이뤄진다.

## 2. 조작

- **탭 = 이동 목표.** 화면 어디든 탭하면 그 지점으로 이동(원형 탭 마커 표시).
- 오브젝트를 탭하면 그 앞까지 경로 이동 후 **자동 상호작용**.
- 고정 이동패드·액션버튼 없음. 하단 바는 메뉴 버튼(인벤토리/상점/건설/요리/광산/퀘스트/도감)만.
- 길찾기는 **격자 A\***(`utils/pathfind.ts`)로 돌길·게이트를 따라 자동 경로 생성.

## 3. 스태미나 & 취침

- 모든 채집 행동은 스태미나 **−1**(행동별 비용은 JSON).
- 스태미나 0: 모든 자동작업 중단, 피로 연출(땀·비네트), "텐트에서 쉬세요" 안내.
- 취침은 **스태미나가 정확히 0일 때만** 가능. 그 외엔 "아직 잘 만큼 피곤하지 않아요."
- 취침 흐름: 텐트 진입 → 암전 → Zzz → 기상 → 스태미나 = 새 최대치, **최대치 +1** → 해 연출 + 축하 팝업 → 즉시 저장.

## 4. 월드 구조 (현재 — Update 6)

- **하나의 연속된 땅.** 바깥 테두리만 바다.
- 내부는 **나무 울타리**로 동일 크기 직사각형 **우리(pen)** 들로 분할. 우리 사이엔 **돌길**.
- 울타리는 이동을 막고, 각 우리는 돌길 쪽 **게이트**로만 진입.
- 콘텐츠 추가 = 동일한 우리 하나 추가.
- 모든 그림은 **정사각 픽셀 도트**로 렌더(곡선·그라데이션 미사용).
- 카메라는 한 우리가 화면에 꽉 차도록 줌, 플레이어를 따라간다.

현재 우리(12칸, 3열 그리드):
`home`(텐트·상점·요리화덕·창고) · `forest`(나무·덤불) · `quarry`(바위) ·
`f_wheat / f_carrot / f_strawberry / f_tomato / f_potato`(작물별 밭) ·
`p_chicken / p_cow / p_apiary`(동물) · `p_mine`(광산 입구).

## 5. 농사 (작물별 밭)

- 수동 파종 없음. 밭(plot)을 사면 **자동 성장 → 인접 시 자동 수확(−1) → 자동 재파종.**
- 작물마다 **전용 밭 우리**가 있고, 그 우리에서 산 밭은 해당 작물로 고정.
- 밭 가격은 그 작물 밭 보유 수에 따라 `base × mult^보유수`로 증가.
- 밀밭은 시작 시 무료 밭 1칸 제공.
- 작물: `wheat, carrot, strawberry, tomato, potato`.

## 6. 동물 / 요리 / 건물

- **동물:** `chicken`(닭→달걀), `cow`(소→우유), `bee`(꿀벌→꿀). 타이머 생산 → 자동 수집(−1).
- **요리:** `bread, pancake, berry_jam, cheese, cheese_toast, veggie_soup`. 제작 타이머 + 레시피 도감.
- **건물:** `tent, shop_stall, cooking_fire, chicken_coop, barn, apiary, storage, farm_sign, mine_entrance`. 건설·업그레이드(시각 변화 + 효과).

## 7. 음식 경제

- 음식은 **먹지 않는다.** 스태미나 회복/증가에 절대 쓰이지 않음.
- 오직 판매·퀘스트·요리 재료로만 사용.

## 8. 퀘스트 / 도감

- 초반 동선을 안내하는 퀘스트 **22개**(이동→벌목→판매→밭 구매→수확→스태미나 소진→취침→광석 채광→건물 건설→요리…). XP 보상 없음.
- 아이템/레시피 도감으로 수집 현황 확인.

## 9. 저장 (localStorage)

- 버전 관리 저장(`SAVE_VERSION`, 현재 **4**). 버전 불일치 시 초기화.
- 10초 자동저장 + 주요 행동/취침 시 저장.
- **오프라인 진행:** 로드 시 작물/동물/요리/업그레이드 타이머만 전진. 자동 수확은 하지 않음 — 플레이어가 직접 가서 수거.

## 10. 사운드 / 이펙트

- 모든 BGM·SFX는 **Web Audio 절차적 합성**(외부 오디오 파일 없음). 상황별 BGM + SFX, 설정(마스터/BGM/SFX/음소거) 영속화.
- 이펙트: 탭 마커, 이동 먼지, 잎/돌조각/반짝임 파티클, 아이템 흡입, 골드·스태미나 팝업, 0일 때 비네트, 취침 페이드+Zzz, 기상 해, 최대치 +1 축하, 건설 반짝임.

## 11. 기술 스택 / 제약

- React 19 + Vite + TypeScript(strict). 추가 npm 의존성 없음.
- TS 제약: `erasableSyntaxOnly`(enum/namespace 금지 → const 객체 + 유니온), `verbatimModuleSyntax`(`import type`), `noUnused*`.
- 데이터 주도: 시스템 코드에 밸런스 하드코딩 금지, 전부 `data/*.json`에서 읽음.
- 렌더: 단일 월드 트랜스폼(`ctx.scale(zoom)` + 정수 라운딩 translate), `imageSmoothingEnabled=false`로 픽셀 지터 제거.

## 12. 디렉터리 개요

```
cozy-island/
  index.tsx            세로 프레임·캔버스·HUD·패널·입력
  game/                Game(루프/상태/렌더) · GameState · Camera · layout · EventBus · rng
  systems/             QuestSystem · SaveSystem
  render/              sprites · art · effects · palette
  audio/               AudioManager (절차적 BGM/SFX)
  utils/               pathfind (격자 A*)
  data/                전 밸런스/콘텐츠 JSON + strings.ko
  docs/                기획서 · 개발플랜
```
